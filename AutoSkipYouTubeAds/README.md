---

## 🚀 Features:

- ⏩ Automatically skips skippable ads
- 🔇 Mutes audio during advertisements
- 📱 Supports regular YouTube videos
- 🎬 Supports YouTube Shorts
- 🌍 Multi-language support

Enjoy a cleaner and less distracting YouTube experience.

---

## ⚙️ Requirements:

To work correctly, the macro requires:

1. Android Accessibility Service enabled
2. MacroDroid UI Interaction accessibility service enabled

**Path:** Android Settings → Accessibility → MacroDroid → UI Interaction

---

## 🔍 How it works:

While YouTube is running, the macro monitors on-screen UI elements and:

- Detects ad screens
- Mutes audio when an ad starts
- Presses Skip buttons automatically when available
- Restores normal behavior after the ad ends

The macro temporarily increases screen-reading frequency to improve detection accuracy and automatically restores the previous value afterward.

---

## 🌐 Language support:

Translations are stored in the `langs` variable.

To add a new language:

1. Copy an existing language entry.
2. Create a new language key (for example: `ru`, `en`, `fr`, `de`, `es`).
3. Translate the values while keeping the original keys unchanged.
