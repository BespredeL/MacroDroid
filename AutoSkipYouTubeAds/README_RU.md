## 🚀 Возможности:

- ⏩ Автоматический пропуск рекламы
- 🔇 Отключение звука во время рекламных вставок
- 📱 Поддержка обычных видео YouTube
- 🎬 Поддержка YouTube Shorts
- 🌍 Поддержка нескольких языков

Смотрите видео без лишних отвлечений и ручного пропуска рекламы.

---

## ⚙️ Требования:

Для корректной работы необходимо включить:

1. Службу специальных возможностей Android
2. Разрешение MacroDroid UI Interaction

**Путь:** Настройки Android → Спец. возможности → MacroDroid → UI Interaction

---

## 🔍 Принцип работы:

Во время работы YouTube макрос отслеживает элементы интерфейса и:

- определяет появление рекламы
- отключает звук во время показа рекламы
- автоматически нажимает кнопки пропуска
- возвращает обычный режим после завершения рекламы

Для повышения точности обнаружения элементов макрос временно увеличивает частоту чтения экрана и автоматически восстанавливает исходное значение после
завершения работы.

---

## 🌐 Поддержка языков:

Все переводы находятся в переменной `langs`.

Чтобы добавить новый язык:

1. Скопируйте существующий перевод.
2. Создайте новый ключ языка (`ru`, `en`, `fr`, `de`, `es` и т.д.).
3. Переведите значения, не изменяя ключи.
