# 📁 Remote Filemanager

🤖 Веб-файловый менеджер для Android на базе HTTP Server в MacroDroid.

Позволяет управлять файлами устройства через браузер - полностью локально, без облака.

---

## 🚀 Возможности

- 🌐 Доступ к файлам через браузер
- 📂 Навигация по папкам
- 📄 Просмотр содержимого директорий
- 📊 Информация о файлах (размер, дата)
- 🗑️ Удаление файлов и папок
- 📁 Создание папок
- 📋 Копирование файлов
- ✂️ Перемещение файлов
- 🌍 Поддержка нескольких языков
- 🔐 Работа в локальной сети (Wi-Fi / hotspot)

### ⚠️ В разработке

- 📥 Загрузка файлов
- 📤 Отправка файлов

---

## ⚙️ Как это работает

Макрос использует встроенный HTTP Server в MacroDroid и превращает устройство в файловый сервер.

Открываешь браузер → вводишь адрес → управляешь файлами.

---

## 🧩 Требования

- 📱 Android устройство
- 📲 Установленный MacroDroid
- 🌐 Одна сеть (Wi-Fi или hotspot)

---

## ⚡ Установка

1. Скачай файл макроса:
   Remote_Filemanager.macro

2. Импортируй в MacroDroid:
   Menu → Import/Export → Import Macro

3. Включи макрос

---

## 🔧 Настройки

### 📌 Основные переменные

| Переменная | Описание | Пример |
|----------|--------|--------|
| port | Порт HTTP сервера | 8080 |
| rootPath | Начальная папка | /sdcard/ |
| allowedFileExt | Разрешённые расширения | jpg,png,txt |
| currentLang | Язык интерфейса | en, ru |

---

### 🔐 Безопасность

| Настройка | Описание |
|----------|--------|
| IP whitelist | Ограничение доступа по IP |

---

## 🌐 Использование

После запуска макроса:

1. Узнай IP устройства (например 192.168.1.100)
2. Открой в браузере:
http://<device_ip>:\<port>/rfm

Пример:
http://192.168.1.100:8080/rfm

---

## 💡 Примеры использования

- Передача файлов между телефоном и ПК
- Доступ к логам и данным
- Управление файлами без кабеля
- Использование телефона как мини NAS

---

## ⚠️ Ограничения

- Производительность зависит от устройства
- Нет HTTPS (только локальная сеть)
- Ограничения MacroDroid
